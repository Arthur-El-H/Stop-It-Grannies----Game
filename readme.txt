22.05.2021 - Stop it Grannies- Game

	What is it about?
This is a 1v1 local multiplayer game. 
You play two brothers on a family festivity trying not to be kissed by dozens of touchy Grannies. But what is that?
You can collect ghettoblasters and use them to push the grannies away or- would you be so nasty? Or onto your brother?

	What is next?
This game is finished in its core version. Some new features may be added + online multiplayer.
Download it under: https://piepscho.itch.io/grannies-stop


Stack used: Unity2D, Visual Studio, C#
